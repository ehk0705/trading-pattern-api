Pine Editor

Cette page sert d’aide pour créer, copier et préparer un script Pine Script avant de le coller manuellement dans TradingView. TradingView ne permet pas à une page externe de remplir directement Pine Editor.

Paramètres rapides

Le script généré doit être collé manuellement dans Pine Editor.

Script Pine généré

Méthode d’utilisation

  1. Choisir l’actif, l’indicateur et l’intervalle.
  2. Cliquer sur « Générer le script ».
  3. Cliquer sur « Copier le script ».
  4. Ouvrir TradingView.
  5. Afficher le graphique souhaité.
  6. Ouvrir Pine Editor dans TradingView.
  7. Coller le script.
  8. Cliquer sur « Ajouter au graphique ».
