Générateur de Pine Script

Cette section transforme les critères choisis dans la page en code Pine Script. TradingView ne permet pas d’insérer automatiquement ce code dans Pine Editor. Il faut donc le copier, ouvrir TradingView, puis le coller manuellement dans Pine Editor.

  1. Choisir un actif.
  2. Choisir un indicateur ou une stratégie.
  3. Choisir un intervalle.
  4. Régler les paramètres avancés : capital, risque, stop loss, take profit, levier, source du prix.
  5. Cliquer sur « Générer Pine ».
  6. Copier le code généré.
  7. Ouvrir TradingView.
  8. Coller le code dans Pine Editor.
Important : le script est généré dans cette page, mais il doit être collé manuellement dans Pine Editor.
